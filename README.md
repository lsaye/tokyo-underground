# tokyo-underground

Project Summary:

	Tokyo Underground is an interactive scrolling website that holds the story of a street racer in modern Tokyo, Japan.  We have used drawings in a comic-book style using boxes that progress as the user scrolls through our site.  We have implemented expressive backgrounds and provoking sounds to help bring our story to life for the reader.  We ideally planned for this project to be a unique piece that we can use in our portfolios using industry-level techniques taught this current and past semester.  Our target audience is mainly people age 16+, interested in street racing, car enthusiasts and suspenseful or thrilling action stories!
